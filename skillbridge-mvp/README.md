# SkillBridge

SkillBridge is an AI-powered personalized learning platform built for a 48-hour hackathon. It combines onboarding, adaptive lessons, dropout-risk interventions, job matching, and an AI assistant.

## Run locally

Backend:

```bash
cd backend
pip install -r requirements.txt
python run.py
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Optional database:

```bash
docker compose up -d
```
